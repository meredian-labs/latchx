# Changelog

All notable changes to Latch will be documented in this file.

## 0.1.0 - Unreleased

Initial `latchx` MVP release candidate.

- Added `latchx audit`, `inspect`, and `run` commands.
- Added pre-execution npm package audit flow.
- Added tarball integrity verification and local cache.
- Added recursive package file scanner and risk scoring.
- Added previous-version diff.
- Added JSON and CI/agent mode.
- Added local policy engine.
- Added `doctor` and cache commands.
- Added demo, policy examples, and dogfood documentation.
